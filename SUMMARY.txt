💡 POR QUÉ ESTO FUNCIONA:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. ResNet50 es SUFICIENTE para COCO
   → ResNet152 es overkill (mucha capacidad sin necesidad)

2. Las capas semánticas DEBEN entrenar
   → Layer3/4 contienen información crucial para segmentación
   → Tu modelo anterior SOLO entrenaba el decoder = muy limitado

3. Learning rates bajos = convergencia lenta
   → Con capas descongeladas, puedo subir LR 50x sin divergencia

4. Data augmentation = mejor generalización
   → 8 tipos diferentes previenen overfitting

5. AMP = GRATIS 1.5-2x speedup
   → Usar float16 donde es seguro, float32 donde importa

6. Warmup scheduler = entrenamiento más estable
   → Evita saltos grandes al inicio
